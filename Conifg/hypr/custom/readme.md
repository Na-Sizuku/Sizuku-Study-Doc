# How to Use

This Config is made by Na-Sizuku.  
This is like i3 styles Hyprland Config, if you was using i3 this config will more easy let you use hyprland.  
This Config will try not use other plugins just use archlinux base packges to make.  
If you want Custome somethings you can check other `.conf` files, and write you want.  

## Auto Start

The Config will Auto Start some program ex. `hyprpaper` `waybar`.  
You can change it in `program-set.conf`, by default you may need install some program thoese are list below.  

## About Waybar

Waybar is Hyprland Status bar recomand install, for custom Waybar you can change it at `/.config/waybar/` jsonc files.  
`top-bar.jsonc` is Waybar top status bar, `bottom-bar.jsonc` is Waybar bottom status bar.

## Install

For use you need install thoese packges below

|Packges Name|Useful|
|:---:|:---:|
|terminator|Terminal|
|dolphin|File manager|
|wofi|Quick start|
|waybar|Top bar&Bottom bar|
|hyprpaper|Desktop wallpaper|

Then download all file for use git clone all file to you locale.  
Backup your Hyprland config and put all you downlaod file to your Hyprland Config file floder.  
Reload Config all just Restart your Hyprland then just enjoy it.
